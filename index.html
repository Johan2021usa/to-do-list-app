Welcome to My To Do List App

You can use this application to create a list of tasks to manage your time properly, follow the next instructions to use the app:

  1. Fill out the Task Description field with your specific task
  2. Add your task to the list by pressing the button Create
  3. Change the task status among pending, completed or ongoing
  4. You can remove from the list the tasks that you have already completed by pressing the button Delete
  5. The color of each task will change according to their status
  6. Your tasks will be displayed whether you close the tab or browser
